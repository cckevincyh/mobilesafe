##安全卫士


这是一个以前学习安卓的时候，跟着视频一起学习做的一个模仿安全卫士的一个简单安全卫士软件。
作为一个安卓学习项目还是不错的。
里面的功能有：

1. 手机防盗
2. 通信卫士
3. 软件管理
4. 进程管理
5. 手机杀毒
6. 缓存清理
7. 高级工具
8. 设置中心

差不多就是这些。
